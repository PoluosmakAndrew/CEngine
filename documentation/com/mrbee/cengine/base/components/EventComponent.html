Implements IEventComponent
Subclasses Graphics2DComponent, SocialComponent, StateComponent

Событийный компонент. Данный компонент имеет набор функций, позволяющих удобно работать с событиями. Этот тип компонента удобно использовать в качестве контроллеров действий или реакции на события определенного объекта в системе.

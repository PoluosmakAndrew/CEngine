Packagecom.mrbee.cengine.base.components
Classpublic class Graphics2DComponent
InheritanceGraphics2DComponent EventComponent EntityComponent Object
Implements IGraphics2DComponent

Графический компонент. Компонент имеет набор функций для работы с графическим контейнером. Для каждого компонента создается новый контейнер Sprite, который размещается на главной сцене приложения в указанном слое. Компонент наследуется от событийного компонента и имеет все его свойства и методы.

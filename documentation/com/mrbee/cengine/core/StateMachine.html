Packagecom.mrbee.cengine.core
Classpublic class StateMachine
InheritanceStateMachine Object

Стейт машина (состояния). Стейт машина позволяет организовать разбивку сущностей на состояния, в которых сущность может иметь разный набор функциональности или же по разному отображаться. Объект стейт машины храниться в самой сущности, при условии ее активации.
